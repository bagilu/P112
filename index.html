P112 是一套可部署於 GitHub Pages、搭配 Supabase 的多單位值班管理系統。第一版不儲存照片、不啟用現場碼 token；學生完成系統簽到簽退後，依單位規則另寄照片給老師或主管作為人工佐證。

學生端

預約正式值班或待命時段、簽到簽退、勾選工作項目、填寫摘要與異常回報、查看個人時數。

管理端

系統管理員建立單位；單位管理員管理成員、排班、工作清單、出勤紀錄、時數與 CSV 匯出。
